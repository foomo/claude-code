## Superpowers Data Storage

- Store all **superpowers specs and plans** under `.claude/` (not under `docs/`).
	- Specs: `.claude/specs/` (e.g., feature specs, API contracts, requirements).
	- Plans: `.claude/plans/` (e.g., implementation plans generated by Superpowers or Claude Code).
- Do **not** store these files in `docs/superpowers/` or in the repository root; keep that directory for human‑oriented documentation only.
